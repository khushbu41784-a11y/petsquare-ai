# PetSquare

This project is a pet services platform with AI chatbot functionality.  
The live website is hosted on Wix: [PetSquare AI Website](https://khushbu41784.wixsite.com/petsquare-ai-1)
.

## Features
- Online Vet Consultation
- Reactivity Therapy for pets
- Thoughtful and calming UI/UX for emotional comfort
- AI-driven pet assistance
 - AI chatbot for pet guidance
- Pet service booking interface

## Note
Due to Wix platform limitations, the source code is not downloadable.  
This repository serves as documentation and reference for the project.
